2009 NEWS 2009-02-20 - IMPROVED MODULE: Uize.Widget.Options

  • Contents
    • 1. New tentativeValueNo State Property
    • 2. Improved values State Property

The Uize.Widget.Options class has been improved with the addition of a new tentativeValueNo state property, and also now supports a values array with elements that are objects (rather than just strings or numbers).

1. New tentativeValueNo State Property

The new tentativeValueNo state property lets you access the index of the value that the user is considering selecting, and serves as a companion to the already existing tentativeValue state property.

2. Improved values State Property

The values state property now supports an array whose elements are objects, where the special name property of each object is used for setting the value state property when a specific option is selected.
