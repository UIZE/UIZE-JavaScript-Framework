2009 NEWS 2009-06-28 - IMPROVED MODULE: Uize.Widget.Options

The Uize.Widget.Options class has been improved with the addition of the optionWidgetClass, optionWidgetProperties, and setValueOnMouseover state properties, and the forAll instance method.

1. The newly added optionWidgetClass state property lets you specify the widget class that should be used for the option buttons.
2. The newly added optionWidgetProperties state property lets you specify values for state properties that should be used when creating option button child widgets.
3. The newly added setValueOnMouseover state property lets you specify whether or not the value of the value state property should be set when the user rests the mouse over an option, instead of the values of the tentativeValue and tentativeValueNo state properties.
4. The newly added forAll instance method is an iterator method that is provided as a convenience and that lets you iterate over all the option buttons for the instance.
