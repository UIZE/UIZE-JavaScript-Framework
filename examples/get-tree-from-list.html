JAVASCRIPT EXAMPLES Get Tree from List
SOURCE

In this example, the Uize.Node.Tree.getTreeFromList static method of the Uize.Node.Tree module is being used to build a tree data object by analyzing the structure of a nested list defined by an HTML ul tag. A tree data object like this can be supplied to a tree menu widget, or can otherwise be used to build UI for navigating to different sections of the document (a contents tree, for example). A tree data object can also be generated by analyzing the occurrence of different CSS classes for section headings at different depths of the document, using the Uize.Node.Tree.getTreeFromPage method (see the Get Tree from Page example).
