JAVASCRIPT EXAMPLES Calendar Controls Wikipedia
SOURCE

In this example, an instance of the Uize.Widget.Calendar class is used to wire up a simple calendar widget that controls the URL for a document in an IFRAME to its right. Use the calendar to choose a date and the IFRAME will load information about what happened on that day in time, courtesy of wikipedia.org. As powerful of a resource as it is, Wikipedia does not - unfortunately - contain reference information for the future. Because of this failing on Wikipedia's part, the calendar widget is configured to not allow selection of dates in the future, through the use of its maxValue state property.
