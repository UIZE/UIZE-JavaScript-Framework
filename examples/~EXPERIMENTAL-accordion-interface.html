SOURCE

In this example, the markup for an accordion interface is being wired up by an instance of Uize.Widget.Options.Accordion. Below the accordion are some links that execute snippets of code to demonstrate the programmatic interface of the widget. Because the accordion widget is a subclass of Uize.Widget.Options, it provides a value set-get property in its interface. The Changed.value instance event for this property can be used to watch for changes in the selected section. The property can be set to programmatically change the selected section. Interact with both the accordion UI and the test links to see how they affect one another.
