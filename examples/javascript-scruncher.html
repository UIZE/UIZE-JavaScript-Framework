SOURCE

The JavaScript Scruncher tool (which makes use of the Uize.Build.Scruncher module) lets you test the scrunching of JavaScript code. Paste some JavaScript into the "SOURCE CODE" tab and then switch to the "SCRUNCHED CODE" tab to see the scrunched version. Switch to the "SCRUNCHER REPORT" tab to see a summary of the file size savings for different facets of the scrunching operation. To start you off, the "SOURCE CODE" tab is set up with the contents of a Uize module. To learn more about how the scruncher works and the options available, consult the explainer All About Scrunching.
