JAVASCRIPT EXAMPLES Slideshow
SOURCE

In this example, an instance of the Uize.Widget.SlideShow widget class is wired up, initialized with an array of photos by setting its slides state property. Each record in the slides array contains title and image properties. The slideshow widget wires up buttons for navigating the slides. When a slide is selected, the value of the slide's record's title and image properties are displayed in the widget's slide_title and slide_image implied nodes, and the current slide number and total number of slides are reflected in the widget's slideNumber and totalSlides implied nodes.
