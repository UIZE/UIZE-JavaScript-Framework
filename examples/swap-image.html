JAVASCRIPT EXAMPLES Uize.Widget.Swap.Image
SOURCE

In this example, an instance of Uize.Widget.Swap.Image has been wired up to a div in the document. This class provides a highly configurable yet lightweight JavaScript animation effect to transition from one image to another. A list of links in the "PRESETS" tab lets you preview all the effect presets defined in the Uize.Widget.Swap.xPresets extension. Click to experience some of the many effects that this widget supports. To fool around with the settings for a preset to create your own swap effects, switch over to the "PARAMS" tab to tweak values and click the "PERFORM SWAP EFFECT" button to preview changes. To understand how the effect settings work, consult the reference for the Uize.Widget.Swap class.
