JAVASCRIPT EXAMPLES Image Cycle
SOURCE

In this example, an instance of Uize.Widget.Swap.Image.Cycle is wired up to provide a simple cycling slideshow of images, with no navigation. The cycleSettings set-get property is set to the Uize.Widget.Swap.presets property (defined in the Uize.Widget.Swap.xPresets module), so each new image shows with a different JavaScript animation transition effect. The buttons below are wired up to call the start and stop instance methods. Click "Start".
