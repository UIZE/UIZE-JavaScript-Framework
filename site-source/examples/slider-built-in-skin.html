JAVASCRIPT EXAMPLES The Built-in Slider Skin
SOURCE

In this example, multiple instances of Uize.Widget.Bar.Slider are being wired up. The markup for all the sliders is not in the page, but is inserted dynamically when the slider widgets are wired up, and is based on the built-in skin supplied by the Uize.Widget.Bar.Slider.xSkin extension to the Uize.Widget.Bar.Slider class. This skin can accept input parameters that control certain aspects of the slider's look.
