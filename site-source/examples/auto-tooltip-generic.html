In this example, an instance of Uize.Widget.AutoTooltip is wired to a bunch of external links that appear within a discussion of the electronica music genre down below (courtesy of Wikipedia). Each link tag has summary info for the link, tucked away inside the tag's title attribute. When you mouse over the links, the summary info is displayed in a table format inside a tooltip that follows the mouse position. The tooltip HTML for the summary table is generated by a JavaScript template that has been pre-compiled to a JavaScript module and has been specified for the instance's html state property. The set of links to wire up with tooltips is obtained using the Uize.Node.find static method, and is specified by setting the instance's nodes state property.

Tucking the summary info for the links into the HTML provides relevancy information for search engines that index the page, but also adds value to the page in the case that JavaScript is disabled, if the JavaScript breaks, or if it just takes long for the JavaScript to load and wire up the behavior. To demonstrate the fallback experience, click on the "UNWIRE AUTO TOOLTIPS" button to unwire the widget. With the widget unwired, mousing over the links gives you the experience without the auto tooltip behavior - just the regular browser tooltips. To go back to decorated tooltips, click the "WIRE AUTO TOOLTIPS" button.
