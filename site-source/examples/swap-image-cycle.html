JAVASCRIPT EXAMPLES Image Cycle
SOURCE

In this example, an instance of Uize.Widget.Swap.Image.Cycle is wired up to provide a simple cycling slideshow of images, with no navigation. The cycleSettings state property is set to the themes defined by the Uize.Widget.Swap.Themes module, so each new image shows with a different JavaScript animation transition effect. The buttons below are wired up to call the start and stop instance methods. Click "Start".
