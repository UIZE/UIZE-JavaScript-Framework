JAVASCRIPT EXAMPLES Sequenced Show
SOURCE

In this example, the Uize.Array.Order.outsideIn method of the Uize.Array.Order package is being used in conjunction with the Uize.Dom.Basics.find method in order to sequence the showing of thumbnail images in a grid. The Uize.Widget.ThumbZoom widget is being used to provide a JavaScript animation zoomed view effect when you click on the thumbnails.
